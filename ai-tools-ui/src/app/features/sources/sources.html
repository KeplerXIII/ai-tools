Разбор ленты

Новые документы создаются с типом, заданным для источника. Глубина (1–30 дней, UTC): отсекаются материалы старше порога после извлечения текста. Без даты — по чекбоксу ниже.

Если отмечено, отдельные шаги ниже игнорируются. Иначе: шаги без зависимости от перевода («Перевод» не считается) ставятся параллельно; тегирование по переводу, аннотация и категоризация совместно с «Перевод» ставятся только после успешного сохранения перевода.

Либо отдельные шаги
По исходному тексту или без перевода

Категоризация без «Перевода»: по текущему тексту в документе (есть перевод в базе — по нему, иначе по оригиналу). Если отмечены и «Перевод», и «Категоризация» — категоризация выполняется после успешного перевода и всегда по переводу.

Только при наличии перевода

Сначала отметьте «Перевод». Эти шаги запускаются автоматически после того, как перевод сохранён в документе (без гонки с очередью перевода).
